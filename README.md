# Advent of Code 2025 - JavaScript/TypeScript

Solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in TypeScript and running with [Bun](https://bun.sh).

## Setup

```bash
bun install
```

## Running Solutions

Run a specific day's solution:

```bash
# Run day 1, part 1 (default)
bun start day1

# Run day 1, part 1 explicitly
bun start day1 1

# Run day 1, part 2
bun start day1 2

# Run day 2, part 1
bun start day2
```

## Project Structure

- `day1.ts`, `day2.ts`, etc. - Individual puzzle solutions
- `inputs/` - Input data for each puzzle
- `index.ts` - Entry point that routes to the correct day and part

## Requirements

- [Bun](https://bun.sh) (runtime)
- TypeScript 5+
